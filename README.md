Powerlifting Program

Simple powerlifting program to keep track of your progress:

It has the big 3 exercises Squat, Bench and Deadlift, as well as slight alternatives to those variatons ex. Pause or Tempo

ICEBOX:

Eventually what it would be able to do is:

1. Let you input your 1 Rep max for each lift
2. Take a % of that 1 Rep max and give it to you for that days workout
3. Allow you to add your own exercises
4. Graphs showing progression of lifts
5. Allow you to add your own Accesories in an accesories category
